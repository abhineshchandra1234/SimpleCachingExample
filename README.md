# SimpleCachingExample

## Overview, Purpose & Features
- this app will fetch data from online and show in recycler view
- it will also store data in room for offline caching

- Recycler View
- Fragments
- Coroutines
- ViewModel && LiveData
- Room
- ViewBinding
